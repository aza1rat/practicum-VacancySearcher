# VacancySearcher
Приложение для поиска вакансий через демонстрационное API, сохранение избранных вакансий в базу данных приложения

## Над проектом работали
https://github.com/aza1rat  
https://github.com/JosephPtn  
https://github.com/Carlmarksjunior  
https://github.com/JohnGudron  

## Особенности
- пагинация в виде бесконечной прокрутки с фоновой загрузкой  
- автоповтор запроса при ошибке сети во время пагинации  
- фильтрация по месту работы, отрасли, ожидаемой зарплате  
- сохранение избранных вакансий в базу данных приложения  

## Технологии и архитектура

### Архитектура
**Архитектурный паттерн** - MVVM  
**DI** - Koin  

### Навигация и интерфейс
**Навигация** - Jetpack Navigation  
**Подход**: Single Activity, Fragments  
**Безопасный доступ к элементам интерфейса** - ViewBinding  

### Слой представления
- ViewModel  
- LiveData  

### Асинхронность
Kotlin Coroutines & Coroutines Flow

### Работа с API
**Сетевые запросы** - Retrofit2  
**Сериализация JSON** - Gson  
**Загрузка и отображение изображений** - Glide  

### Хранение данных
**Сохранение избранных вакансий** - Room  
**Сохранение настроек фильтрации** - SharedPreferences  

### Инструменты анализа кода
**Инструмент статического анализа кода, запускался при pull request** - Detekt
